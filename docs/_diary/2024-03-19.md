# 何もかもがクソ深い国
夕方の便で羽田に着いた。以前マズいことで有名だったはずの機内食はとても美味しく、もはや空中含めどこでもご飯が美味しいクソ深い世界になってしまったことが判明した。
帰国はがら屋に行き、ラーメンと餃子を食べた。想像より何倍も美味いと上げ続けた期待値を裕に超える味でさらに深くなってしまった。
あと日本語の車内アナウンス、笑ってしまうほど深い
